Quick Install Guide¶

Prerequisites¶

SmallWorld is supported by Python (>=3.10) on both Windows and Linux and is installed with pip. Both Python and pip must be installed on your system before attempting to install SmallWorld.

Installation¶

SmallWorld is currently only installable from source - packages will be available in PyPI with our first major release!

For now, to clone the source code, run:

git clone git@github.com:smallworld-re/smallworld.git

To install SmallWorld with pip from the source repo, run:

pip install .
